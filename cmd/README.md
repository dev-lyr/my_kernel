# 一 概述:

## (2)信息查询:
- sar
- **uptime**: 显示当前时间, 系统开机时间, 当前登录用户数, 以及前1,5,15分钟内的平均负载情况. 平均负载:特定时间内运行队列中进程的平均数, 每个CPU核不超过3代表系统负载良好, 例如:双核CPU, uptime最后显示的不超过6.
